JS Data Types

JS variables can hold many data types: numbers, strings, objects an more.

The Concept of Data Types

In programming, data types is an important concept.

To be able to operate on variables, it is important to know something about the type.

JS evaluates expressions from left to right. Different sequences can produce different results.

JS Types are Dynamic

JS has dynamic types, this means that the same variable can be used to hold different data types.

Js String

A string is a series of characters like 'Hello'.

JS Numbers

JS has only one type of numbers. Numbers can be written with or without decimals

Extra large numbers can be written with scientific (exponential) notation.

var y = 123e5; // 12300000
var y = 123e-5; // 0.00123

JS Booleans

Booleans can only have two values: true or false.

JS Arrays

JS arrays are written with square brackets, array itmes are separated by commas.

Arrays indexes are zero-based, which means the first item is [0], second is [1], etc..

JS Objects

JS Objects are written with curly braces { }.

Object properties are written as name:value pairs, separated by commas.

var person = {
firstName:"Hans",
lastName:"Mustermann", age:60,
eyeColor:"blue"
};

The typeof Operator

We can use the JS typeof operator to find the type of JS variable

Undefined

In JS a variable without a value has the value of undefined. The type is also undefined.

Empty Values

An empty value is not undefined, it has both legal value and a type ('string').

Null

In JS null is 'nothing'. It is something that does not exist.

Unfortunately in JS the data type of null is an object.

It is considered a bug in JS that typeof null is an object, it should be null.

We can empty an object by setting it to null and also by setting it to undefined.

Difference Between Undefined an Null

null and undefined are equal in value but different in type.

Primitive Data

A primitive data value is a single simple data value with no additional properties and methods.

The typeof operator can return one of these primitive types:

  • string
  • number
  • boolean
  • undefined

Complex Data

The typeof operator can return one of two complex types:

  • function
  • object

The typeof operator returns "object" for objects, arrays and null.

The typeof operator does not return "object" for functions.
