JS Syntax

Js syntax is the set of rules how JS programs are constructed.

var x, y, z; // Declare Variables
x = 5; y = 6; // Asign Values
z = x + y; // Compute Values

JS Values

The JS syntax defines two types of values:

  • Fixed values
  • Variable values

Fixed values are called Literals.

Variable values are called Variables.

JS Literals

The two most important syntax rules for fixed values are:

  1. Numbers - written with our without decimals
  2. Strings - are text within double or single quotes

JS Variables

In programming language variables are used to store data values.

var x;
x = 9;

JS Identifiers

Identifiers are names. In JS the first letter must be a letter or (_) or $ sign.

JS is CAse Sensitive

All JS identifiers are case sensitive.

JS and Camel Case

Historicaly programmers have used different ways to join multiple words into one variable name.

Hyphens

Hyphens are not allowed an are reserved to substraction.

first-name

Underscore

first_name

Upper Camel Case (Pascal Case)

FirstName

Lower Camel Case

firstName
