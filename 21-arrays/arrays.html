What is an Array?

An array is a special variable which can hold more than one value at a time.

An array can hold many values under one name and we can access the values by referring to an index number.

Creating an Arary

Using an array literal is the easiest way to create a JS Array.

var array_name = [item1, item2, item3, ...];

Using the JS Keyword new

There is no need to use new Array(), the literal method above will do the job.

Arrays are Objects

Arrays are a special type of objects. The typeof operator in JS returns 'object' for arrays.

JS arrays are best dscribed as arrays, they use numbers to access its 'elements' in the example cars[0] returns Nissan.

Arrays Elements Can Be Objects

JS variables can be objects, arrays are special kind of objects.

We can have objects in an array, we can have functins and we can have arrays in an array.

myArray[0] = Date.now;
myArray[1] = myFunction;
myArray[2] = myCars;

Array Properties and Methods

Real strength of JS arrays are the built-in array properties and methods:

var x = cars.length;
var y = cars.sort();

The length Property

The length property of an array returns the length of an array.
