Converting Arrays to Strings

The converting JS method toString()converts an array to a string of array values.

The join() method also joins all array elements into a string.

It behaves just like toString(), but in addition you can specify the separator.

Popping and Pushing

Whe working with arrays it is easy to add and remove elements from an array.

Popping items out from an array, and pushing items into an arrays.

Popping

The pop() method removes the last element from an array.

Shifting Elements

Shifting is equivalent to popping, working on the first element instead of the last.

The shift() method removes the first array element and "shifts" all other elements to lower index.
