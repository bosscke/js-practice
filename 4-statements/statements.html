JavaScript Programs

A program is a list of 'instructions' to be executed by a computer, in a programming language these instructions are called statements. JS program is a list of statements.

JS Statements

JS statements are composed of:

  • Values
  • Operators
  • Expressions
  • Keywords
  • Comments

Semicolons ;

Semicolons ; separate JS statements.

JS White Space

JS ignores white space, it is a good practice to put space around operators.

JS Line LEngth and Line Breaks

Avoid code lines longer 80 characters.

JS Code Blocks

JS statements can be grouped together in code blocks, inside curly brackets {...}.

The purpose of code blocks is to define statemetns to be executed together.

JS Keywords

JS statements often start with a keyword to identify the JS action to be performed. Here a lsist

Keyword Description
break Terminates a switch or loop
continue Jumps out of a loop and starts at the loop
debugger Stops the execution of JS an repeats the block while condition is true
do .. while Executes a block of statements
for Marks a block of statement to be executed as long as condition is true
function Declares a function
if .. else Marks a block of code to be executed, depending on a condition
return Exits a function
switch Marks a block of statement to be executed, depending on different cases
try ...catch Implements error handling to a block of statements
var Declares a variable
