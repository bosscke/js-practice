Using let and const (ES6)

Before 2015 using var keyword was the only way to declare JS variable. The version of 2015 allows the use of const key word to define a variable that cannot be reasssigned and let keyword to define a variable with restricted scope.

Like in Algebra we use variables in expressions.

JS Identifiers

  • Names must begin with letters
  • Names can contain digits, underscores, $ signs
  • Names can also begin with $ and _
  • Reserved keywords cannot be used

Declaring JS Variables

After declaration the variable has no value (undefied).

Value = undefined

Re-Declaring JS Variables

JS Dollar Sign $

In JS library jQuery the main function $ is used to select HTML elements. In jQuery $("p") means 'select all p elements'.

JS Underscore (_)

Convention among professional programmers is to use it as an alias for "private (hidden) variables".
